# GitHub Pagesへの反映

既存の `seam-wild-links` リポジトリを使う場合：

1. このフォルダの `index.html` をダウンロードする
2. GitHubで対象リポジトリを開く
3. 既存のルート直下 `index.html` を開く
4. EditまたはDeleteで古い内容を置き換える
5. 新しい `index.html` をルート直下へアップロードする
6. `Commit changes` を押す
7. Pagesの公開処理が完了してからサイトを再読み込みする

`index.html`単体にゲーム本体、ピクセル描画、音響生成を内包しています。追加の画像フォルダは不要です。

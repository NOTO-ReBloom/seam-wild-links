# SEAM: THREAD MEADOW V4 修正内容

## 公開版でキャラクターが表示されなかった原因

公開リポジトリでは `bear.svg`、`rabbit.svg`、`world_0.svg` がルート直下に置かれていた一方、`index.html` は `assets/characters/bear.svg`、`assets/characters/rabbit.svg`、`assets/worlds/world_0.svg` を参照していました。参照先が存在しないため、タイトル画面・説明画面・ゲーム画面の画像読み込みが失敗していました。

## V4の修正

- 外部画像ファイルへの依存を完全撤廃
- クマとウサギをCanvas上で直接描画
- タイトルと説明画面のキャラクターもHTML内のSVGへ内包
- `index.html` 一つだけで公開可能
- クマとウサギの歩行、傾き、倒れ、能力オーラを追加
- 布の繊維、縫い目、丘、草、糸巻き、リボン、巨大織機を背景へ追加
- 4ステージごとに背景のランドマークを変更
- 壁、危険物、縫い目、布、風留め、フック、出口、ボスを描き直し
- ヒットストップ、画面揺れ、粒子、足音を調整
- ウィンドウがフォーカスを失った際のキー入力残留を防止

## GitHub Pagesへの反映

リポジトリ直下の古い `index.html` を、このフォルダの `index.html` に置き換えるだけです。追加画像やassetsフォルダは不要です。

# ゲームの幅 — V14追加内容

## 全体構成
- 物語キャンペーン：5世界 × 3幕 = 15幕
- キュレーター試練：5種類
- 合計：20のプレイ体験

## キュレーター試練

### 1. 光のリレー / RELAY
ブルーノとミミに交互に指定される8つの光輪を順番に通過します。

- 担当キャラクターが毎回切り替わる
- 光輪通過で時間、顔料、速度を回復
- 相手を待ちながらも止まりすぎない高速協力

### 2. 生きた顔料 / ESCORT
二人の間に浮かぶ顔料球をゴールへ運びます。

- 二人が近くにいる間だけ進む
- 離れると停止し、少し後退する
- ルート選択と距離管理を同時に行う

### 3. 季節の二重奏 / DUET
離れたANCHOR印とBRUSH印へ同時に立ちます。

- 4組を順番に起動
- 成功するたび春と冬の地形が反転
- 移動、位置合わせ、合図が中心

### 4. 修正機関 / BOSS
三段階の協力ボス戦です。

1. ブルーノが近くでANCHORを維持
2. 弱点が一定時間だけ露出
3. ミミがチャージ済みの筆跡で弱点を横切る
4. 修正波を回避しながら3層を破壊

### 5. 星夜逃走 / CHASE
迫る消去波から逃げながら、3つの星灯台を点灯します。

- 二人が同じ灯台の近くでQとEnterを同時長押し
- 点灯すると闇を押し戻し、星足場が一定時間現れる
- 走る区間と短い同時操作を交互に配置

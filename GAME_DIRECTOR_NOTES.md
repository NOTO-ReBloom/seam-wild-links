# GAME DIRECTOR NOTES V3

## 中心となる面白さ
相手の位置を利用し、失敗しかけた相手を救いながら、一本の糸を移動・運搬・追跡・ボス攻略へ転用する。

## 4ステージの感情曲線
1. 発見：安全な場所で糸の距離を理解する。
2. 協調：二人持ちで歩調を合わせ、役割交代する。
3. 圧力：追跡下で救助を要求する。
4. 解放：学んだ糸の使い方を大型ボスへ集約する。

## 実装原則
- 操作は移動＋能力のまま増やさない。
- 片方が倒れても、相手の操作で復帰できる。
- 成功時は音、停止感、粒子、画面揺れを重ねる。
- 各ギミックは移動・協力・救助・見せ場の複数用途を持つ。

# MASTERPIECE DIVE V16 — FIRST DIVE GUIDANCE

GitHub Pagesへは、このフォルダ内の `index.html`、`404.html`、`.nojekyll` をそのまま配置してください。

## V16の主な改善

- ACT 1開始前に4ページのドット調ビジュアルガイドを自動表示
- 一台のPCを二人で操作する担当分担を明示
- ANCHORのQ長押し、BRUSHのEnter長押しを図解
- 最初の面を「移動→ジャンプ→固定＋描画→橋を渡る→二人でゴール」の5段階に再設計
- キャラクター付近、能力地点、構図印、橋、門へキー表示と矢印を配置
- 構図完成後は消えない一本の安全な橋が出現
- Hで状況別ヒント、Rでチェックポイント復帰、Shift+Rで幕の最初から再開
- 図解はタイトル、美術館、ポーズ画面から再表示可能

単一HTML構成で、外部画像参照はありません。

# MASTERPIECE DIVE V14 — CURATOR TRIALS

## 起動
`index.html`をブラウザで開きます。GitHub Pagesでは`index.html`と`assets`フォルダを同じ階層へアップロードしてください。

## 収録内容
- 5つの名画世界
- 15幕のストーリーキャンペーン
- 5つのキュレーター試練
- 記憶の残響、名画の欠片、スコア、ランク、進行保存

## 操作
### ブルーノ
- A / D：移動
- W：ジャンプ
- Q：ANCHOR、空中ではANCHOR SLAM、救助

### ミミ
- ← / →：移動
- ↑：ジャンプ
- Enter：BRUSH、滑空、救助

### 共通
- QとEnter同時押し：RESONANCE 100%時にMASTERSTROKE
- R：現在の幕または試練を再開始
- H：ヒント
- Esc：一時停止

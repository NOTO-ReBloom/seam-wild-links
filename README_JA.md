# SEAM: WOVEN WORLDS V7

1台のPC・1つのキーボードで遊ぶ、2人協力の横スクロール・ピクセルアドベンチャーです。

## 起動方法

`index.html` をGoogle ChromeまたはMicrosoft Edgeで開いてください。

- サーバー不要
- 外部ライブラリ不要
- 外部画像不要
- データベース不要
- GitHub Pages対応

## 基本目的

全コース共通で、2匹そろって画面右端にある「世界の門」へ到達するとクリアです。
各コースには任意取得の大型収集物が3個あります。

## 操作

### PLAYER 1
- A / D：移動
- W：ジャンプ
- Q：章ごとの能力

### PLAYER 2
- ← / →：移動
- ↑：ジャンプ
- Enter：章ごとの能力

### 共通
- Esc：一時停止
- R：コースを最初から再開

## 収録内容

- 全5章
- 全20コース
- 10体のプレイアブル動物
- 章ごとに異なる協力能力
- 各コースに収集物3個
- 中間地点、スコア、ランク、進行保存

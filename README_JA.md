# SEAM: TENSION TRIALS

一台のPCと一つのキーボードで遊ぶ、2人協力型ピクセル物理パズルです。

大きなクマのブルーノ（重量3）と、小さなウサギのミミ（重量1）は、伸縮する一本の糸で結ばれています。重量差、慣性、張力、回転、反発を読み、二人で高難度の試験を突破します。

## 起動方法

`index.html` をChrome、Edge、Firefoxなどのブラウザで開いてください。サーバー、インストール、外部ライブラリは不要です。

GitHub Pagesへ公開する場合も、リポジトリ直下へ `index.html` を置くだけで動作します。

## 操作

### PLAYER 1 — BRUNO

- `A` / `D`：移動
- `W`：ジャンプ
- `Q`：踏ん張る／操作する

### PLAYER 2 — MIMI

- `←` / `→`：移動
- `↑`：ジャンプ
- `Enter`：踏ん張る／操作する

### 共通

- `H`：段階式ヒント
- `R`：ステージを最初からやり直す
- `Esc`：一時停止

## ゲームの特徴

- 480×270の低解像度Canvasを整数拡大する、統一されたピクセルアート表現
- クマ3：ウサギ1の重量差を使った物理解法
- 伸縮糸の張力、フックへの経路変更、踏ん張りによる固定
- シーソー、滑車、振り子、慣性、壊れる床、可動リフト、上昇する危険領域
- 解法を考えるパズル性と、実行精度を要求するチャレンジ性
- 全5試験、順次解放、ベストタイムとランク保存
- Web Audio APIによるチップ音響

## 試験

1. **重さを読む** — 重量差とカウンターウェイト
2. **滑車の谷** — 糸の経路、固定、振り子
3. **慣性の庭** — シーソー発射と衝突速度
4. **張力の回廊** — 壊れる足場と上昇する危険領域
5. **心臓織機** — 重量、滑車、張力、発射を統合した最終試験

## 保存

進行状況とベストタイムはブラウザのLocalStorageへ保存されます。保存が許可されていない環境でも、ゲーム自体は動作します。
